# TamperMonkey Scripts

This is a collection of personal scripts that I wrote and use while browsing the internet.

## Scripts

- [twitter-ad-highlight](./scripts/twitter-ad-highlight.js): Adds a really ugly red background color to every ad post, so you know which ones to just scroll past, without wasting any time on procrastination:
  ![twitter-ad-highlight-recording](https://i.ibb.co/fGYB6QFk/ezgif-2ec569182e7ce410.webp)
- [export-iframe-html](./scripts/export-iframe-html.js): Adds a button _adjacent_ to every `<iframe>` element. When clicked, it downloads the contents of the DOM inside the iframe as HTML file.
  ![export-iframe-html-screenshot](https://i.ibb.co/KcZ2prRm/Screenshot-2026-03-31-at-20-08-06.png)
